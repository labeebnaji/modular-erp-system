# اختصارات لوحة المفاتيح لوحدة نقاط البيع (POS)

توضح القائمة التالية اختصارات لوحة المفاتيح الرئيسية المستخدمة في وحدة نقاط البيع ووظائف كل منها.

| الاختصار       | الوظيفة                                   | الوصف                                                              |
| :------------- | :---------------------------------------- | :----------------------------------------------------------------- |
| `F1`           | جديد                                      | يمسح جميع المدخلات ويبدأ معاملة مبيعات/مرتجعات جديدة.                |
| `F2`           | حفظ                                       | يحفظ فاتورة المبيعات/المرتجعات الحالية.                             |
| `F8`           | حذف                                       | يحذف بنود فاتورة المبيعات/المرتجعات الحالية.                         |
| `F7`           | طباعة                                     | يطبع إيصال فاتورة المبيعات/المرتجعات الحالية.                      |
| `F6`           | معاينة                                    | يعاين إيصال فاتورة المبيعات/المرتجعات (وحدة نائب حاليًا).        |
| `F5`           | بحث                                       | يفتح مربع حوار للبحث عن فواتير المبيعات/المرتجعات (وحدة نائب حاليًا). |
| `PageUp`       | الأول                                     | يتنقل إلى الفاتورة الأولى في سجلات المبيعات/المرتجعات.             |
| `Left Arrow`   | السابق                                    | يتنقل إلى الفاتورة السابقة في سجلات المبيعات/المرتجعات.            |
| `Right Arrow`  | التالي                                    | يتنقل إلى الفاتورة التالية في سجلات المبيعات/المرتجعات.            |
| `PageDown`     | الأخير                                    | يتنقل إلى الفاتورة الأخيرة في سجلات المبيعات/المرتجعات.            |
| `F3`           | تعليق                                     | يعلق فاتورة المبيعات الحالية للاسترداد لاحقًا.                     |
| `F4`           | استرداد معلقة                             | يعرض مربع حوار لاسترداد الفواتير المعلقة.                          |
| `Escape`       | خروج                                      | يغلق النافذة الحالية.                                             |
| `Ctrl + Enter` | إضافة صنف                                 | يضيف الصنف المحدد بالكمية والسعر إلى جدول الفاتورة.              |
